
KNSA-shooting-target - v10 KNSA v5 - Deleted black circle annot - added points to labels
==============================

This dataset was exported via roboflow.ai on June 1, 2022 at 10:30 AM GMT

It includes 176 images.
Bulletholes are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 416x416 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* 50% probability of horizontal flip
* 50% probability of vertical flip
* Random brigthness adjustment of between -25 and +25 percent
* Random Gaussian blur of between 0 and 1 pixels
